# 铁巢重炮射表计算器

一个用于计算炮击仰角的在线工具，支持多弹道并行计算和历史记录。

## 功能特性

### 核心计算
- **仰角计算**：根据距离和装药量自动计算炮击仰角
- **仰角限制**：最大仰角限制为 60°
- **自动装药推荐**：根据距离自动推荐最小所需装药量
- **强制装药6**：可选择忽略装药限制，强制使用装药6计算

### 装药配置
| 装药 | 倍率 | 最大可用距离 |
|------|------|-------------|
| 1 | 12.0°/km | 5 km |
| 2 | 6.0°/km | 10 km |
| 3 | 4.0°/km | 15 km |
| 4 | 3.0°/km | 20 km |
| 5 | 2.4°/km | 25 km |
| 6 | 2.0°/km | 100 km |

### 多弹道支持
- 可同时管理多个弹道计算
- 每个弹道独立展开/收起
- 弹道状态自动保存

### 历史记录
- 自动保存计算历史到 localStorage
- 支持从历史记录恢复弹道
- 最多保存 50 条记录

## 使用方法

1. 打开 `铁巢重炮射表计算器.html` 文件
2. 输入目标距离（单位：公里）
3. 程序自动推荐合适的装药量
4. 可手动选择其他装药或勾选"强制使用装药6"
5. 查看计算得出的仰角

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需安装，浏览器直接运行
- localStorage 本地存储

## 键盘快捷键

- **回车**：输入距离后按回车收起当前标签
